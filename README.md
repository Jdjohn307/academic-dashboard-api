# academic-dashboard-api
Academic Dashboard API/Backend in Rails/PostgreSQL

* Ruby version
 ruby 3.4.4 (2025-05-14 revision a38531fd3f) +PRISM [x86_64-linux]

* System dependencies
 Ruby version: 3.4.4
 psql (PostgreSQL) 12.22 (Ubuntu 12.22-0ubuntu0.20.04.4)
 Rails version: 8.0.2
 Web Server: Puma (>= 5.0)

* Configuration
  Just run a 'bundle install' and it should install everything

* Database creation
 Running db:migrate should create the database

* How to run the test suite
 bundle exec rspec
